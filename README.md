# bytesfest.github.io

## Tentang
Projek ini hanya percobaan untuk meningkatkan Developer Experience dalam mengembangkan sumber kode situs [Bytesfest](https://github.com/bytesfest/bytesfest.github.io)

## Setup
### Kebutuhan
`NodeJS`

### Instalasi

```sh
npm install
# atau
pnpm install
```

### Menjalankan
```sh
npm dev
```

### Build
```sh
npm build
```
